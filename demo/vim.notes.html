 193 Move
 194 
 195   • Moving around is one crucial part of the flow when using Vim
 196   • Avoid using arrow keys to move
 197   • Avoid move in Insert mode
 198   • Better to avoid using repeated h/j/k/l to move
 199   • Most moves can be repeated by count
 200
